# mhw3d-detection
A scalable xarray/dask framework for detecting 3D marine heatwaves.

Note: to use in this current form, you need to load the code as a package:
To do this we do a pip install within our repository (on your local machine/cluster), so:

cd path/to/mhw3d-detection
conda activate mhw3d
pip install -e .
